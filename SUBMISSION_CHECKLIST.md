# Submission Checklist

Use this before final handoff.

## Required

- [ ] Push source code to a public GitHub/GitLab repository
- [ ] Ensure `README.md` explains:
  - [ ] feature set
  - [ ] tech stack
  - [ ] local run steps
- [ ] Record and attach demo video (Loom/YouTube/file) showing:
  - [ ] wall calendar UI
  - [ ] start and end date selection
  - [ ] in-range highlighting
  - [ ] notes entry/persistence
  - [ ] desktop view
  - [ ] mobile responsive view

## Optional (Recommended)

- [ ] Deploy app (Vercel/Netlify)
- [ ] Add live demo URL to README and submission message

## Pre-submit Sanity

- [ ] `npm run lint` passes
- [ ] `npm run build` passes
- [ ] App loads cleanly in browser without runtime errors
