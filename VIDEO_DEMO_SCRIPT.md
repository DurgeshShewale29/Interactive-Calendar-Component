# Video Demo Script (2-3 minutes)

Use this as a speaking guide while recording.

## 1) Intro (10-15s)

"This is my Interactive Calendar Component built with Next.js and React.  
It is inspired by a physical wall calendar and includes range selection, notes, and responsive behavior."

## 2) Layout Walkthrough (20-30s)

- Point out hero image and wall-calendar styling.
- Mention visual hierarchy between image, notes, and date grid.
- Highlight month/year display and navigation controls.

## 3) Date Range Selection (30-45s)

- Click one date as start.
- Click another date as end.
- Show in-between date highlighting.
- Clear the range and repeat once with reversed order (end date earlier than start) to show auto-correction.

## 4) Notes Feature (20-30s)

- Type a monthly memo in the notes section.
- Move to another month and back.
- Show that notes persist via `localStorage` and are month-specific.

## 5) Responsive Behavior (30-45s)

- Open browser device toolbar.
- Show desktop layout.
- Switch to a mobile width and demonstrate stacked layout.
- Tap dates and notes field to confirm touch usability.

## 6) Close (10s)

"The solution is strictly frontend, with no backend/API, matching the assignment scope.  
Thank you for reviewing."
